ESSE CODIGO PEGA O ELEMNTO "A" E COMPARA COM O ELEMENTO "b"
OS ELEMENTO DE "B" PRESENTE EM "A" SÃO REMOVIDOS E UM NOVO ARRAY É CRIADO COM OS NUMEROS RESTANTES

A[1, 2, 3], B[1] = [2,3]
A[1, 2, 3], B[1,2] = [3]
A[1, 2, 3, 5], B[] = []
A[], B[1] = []

SE A OU B ESTIVEREM VAZIOS O NOVO ARRAY DEVE RETORNAR VAZIO

OS ARQUIVOS DO NODE SERVEM APENAS PARA O USO DO NODEMON PARA AUTOMATIZAR AS ALTERAÇÕES DO CÓDIGO
